Keyboard Shortcuts

Formatting

  • Ctrl+B - Bold
  • Ctrl+I - Italic
  • Ctrl+U - Underline
  • Ctrl+S - Open save dialog

Headings & Paragraphs

  • Ctrl+0 - Normal paragraph
  • Ctrl+1 - Heading 1
  • Ctrl+2 - Heading 2
  • Ctrl+3 - Heading 3
  • Ctrl+4 - Heading 4
  • Ctrl+Shift+J - Previous H1 heading
  • Ctrl+Shift+K - Next H1 heading

Lists & Blocks

  • Ctrl+G - Unordered list
  • Ctrl+Shift+G - Ordered list
  • Tab - Indent list item
  • Shift+Tab - Outdent list item
  • Ctrl+Q - Blockquote
  • Shift+Enter - Line break

Text Color & Styles

  • Ctrl+D - Apply last text color
  • Ctrl+Shift+D - Remove text color
  • Ctrl+H - Apply last highlight color
  • Ctrl+Shift+H - Remove highlight

Undo/Redo

  • Ctrl+Z - Undo
  • Ctrl+Y or Ctrl+Shift+Z - Redo
